### 변수

> 메모리(RAM)에 시간에 따라 **변하는 값**을 기록하기 위한 공간

```java
int sum = 0; // 언제든지 변하는 값 넣을 수 있음
```

> 값에 의미를 부여하여 코드 의미 파악 용이
